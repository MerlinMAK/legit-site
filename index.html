Formation as code

Spin up a company in one call.

Form the entity, procure the EIN, assign a registered agent, and KYC the human at the root — in a single API call. Issue a verifiable certificate, then check or revoke standing programmatically.

Embed it behind your own product, or send customers to a clean self-serve dashboard. Same primitives either way.

REST + SDKsWebhooksMCP server for agentsNo-code dashboard
  formation.ts
import Legit from "@legit/sdk";
const legit = new Legit(process.env.LEGIT_API_KEY);

// one call → entity + EIN + registered agent + KYC
const co = await legit.entities.create({
  type: "series",        // llc | series | standalone
  jurisdiction: "DE",
  name: "Orion Labs",
  owner: { verify: "persona" },
});

co.id;              // "LGT-7421"
co.certificate_url; // verifiable cert of formation
co.registered_agent;// "DE" · all 50 states
await legit.entities.checkStanding(co.id);
